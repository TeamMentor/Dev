ABOUT TEAM Mentor

TEAM Mentor is TEAM Academy’s eKnowledge Base. TEAM Mentor is an interactive Application Security library with thousands of articles on how to prevent vulnerabilities during application development. Use TEAM Mentor to find guidance for implementing the application security controls that are relevant to your particular application in your development language. With optional plugins to the most popular Static and Dynamic code analysis tools, TEAM Mentor provides users with quick and easy access to comprehensive security guidance that is accurate and relevant to specific code security questions. Integrating security scanning and guidance into a development workflow ultimately results in quicker production of more secure and stable applications.


TEAM MENTOR SAST/DAST PLUGINS

TEAM Mentor integrates with many SAST/DAST plugins. When a vulnerability finding is clicked, TEAM Mentor guidance is displayed to give developers vulnerability descriptions and impact, remediation checklists and sample code in the relevant programming language.

Available Plugins interface with:

  • HP Fortify® SCA 4.0
  • Checkmarx Cx Suite
  • Microsoft CAT.net
  • Microsoft Visual Studio
  • Eclipse
